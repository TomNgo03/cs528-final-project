# Data

Raw gesture recordings are saved under:

```text
data/raw/{gesture_name}/{gesture_name}_{index:03d}.csv
```

For pipeline testing before real glove collection, synthetic raw data can be generated with:

```bash
python scripts/generate_synthetic_dataset.py --count 100 --overwrite
```

Synthetic data is useful for testing preprocessing, training, and demo code, but real MPU6050 glove recordings should be used for final model accuracy.

Processed feature tables are generated by:

```bash
python scripts/preprocess_data.py
```

Generated files:

- `data/processed/features.csv`
- `data/processed/labels.csv`
